## Technologies and libraries
* Html
* Css
* Javascript
* JQuery

## Workflow
**#1** Hide the nav menu with a Css `media query`.  
**#2** Show the hamburger menu at `click` on the top-right icon.  
**#3** Use `show` and `hide` to manipulate the DOM.
